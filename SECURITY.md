# Security Policy

## Supported Versions

Currently, Kimi CLI only provides security support for the latest version.

## Reporting a Vulnerability

Please report a vulnerability via the [lihan0705/kimi-cli-plus - Security](https://github.com/lihan0705/kimi-cli-plus/security) page, or open an [issue](https://github.com/lihan0705/kimi-cli-plus/issues) if it can be published to public.
